# WorldCapp

The first project I've ever created all by myself. 

It is a super simple app, showing a list of all FIFA World Cup events with details of each tournament.

## What I used?

- UITableViewController to hold a list of all events
- Structs responsible for holding the data
- UIImage extension, using Core Graphics to draw a nice thin border over the countries flags
- Layout programatically
